## 基于京东Taro框架的小程序app
### 主要技术 React+Taro
> 微信小程序出了很久，一直没有做过类似的东西，最近看了下小程序文档，不vue不react的感觉，事件绑定写法尤其不习惯，心态崩溃前发现了`Taro`，可以让开发者用react来开发多端应用，对应还出了UI框架`Taro-UI`，整体用下来，对小程序的适配基本上很完美，`Taro-UI`部分组件的bug提出issue后也很快看到了官方的更新，对react开发者相当友好了，本项目也是基于`Taro`开发。
